Introducción

Diseño Instruccional: Objeto Virtual de Aprendizaje para Lenguaje: "El abecedario"

Se describe el diseño de un OVA para el aprendizaje de lenguas castellana, se trabajará el abecedario. Basados en nombrar las letras del abecedario, lectura y escritura de las letras etc.
las partes del cuerpo
Elementos de contextualización:
Área de Conocimiento: Lengua castellana
Tema: El abecedario

Objetivo de Aprendizaje

Identifico las letras del abecedario de forma visual para a partir de allí establecer relación con objetos y palabras

Competencias

  • Reconocimiento de la lengua de iconos
  • Lectura y escritura
  • Escrituras de forma para representa la realidad
  • Noción del sistema lingüístico, sus reglas y ordenación.
